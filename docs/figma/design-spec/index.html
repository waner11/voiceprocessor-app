Serving Locally

To view the full design spec in your browser, serve this directory with any static file server:

cd docs/figma/design-spec && python3 -m http.server 3333 cd docs/figma/design-spec && npx serve -p 3333

Then open http://localhost:3333 in your browser. Use Figma's Code-to-Canvas to capture screens.
